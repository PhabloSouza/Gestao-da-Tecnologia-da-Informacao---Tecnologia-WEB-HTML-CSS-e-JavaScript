HD ou SSD: qual armazenamento é melhor?

Disco rígido é mais lento, mas costuma ter maior capacidade e preço mais baixo, mas quem precisa de performance deve investir em um notebook com SSD. Antes de escolher o seu novo notebook, você vai se deparar com muitos modelos com diferentes tipos de armazenamento de arquivos, o que também impacta no preço. Existem pontos positivos e negativos para o HD tradicional e para o SSD.
